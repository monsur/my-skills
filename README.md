# my-skills

Personal Claude Code skills. Symlinked from `~/.claude/skills`.

## Skills

- **`/idea <text>`** — Capture an idea to `~/Documents/Projects/_projects/ideas.md`
- **`/note <text>`** — Jot down a note, insight, or aha moment during a session
- **`/recap`** — Generate a Bluesky thread recap of the session, combining saved notes with session context. Output is formatted for [Threadweaver](https://github.com/monsur/threadweaver).

## Setup

```bash
git clone https://github.com/monsur/my-skills ~/Documents/Projects/my-skills
ln -s ~/Documents/Projects/my-skills ~/.claude/skills
```
